Sizing#

as we have titles, we can change their sizer by: is-size-x like follow:

<h1 class="title is-size-4">Bulma?</h1>

Collumns#

is-multiline

is-0 removes the gap over a grid

is-mobile keeps the size over mobile

</figure>
